About the Website

Fake News Detection System is a dedicated platform committed to identifying and debunking fake news stories circulating online. Our team of experts uses advanced technology and research to verify the authenticity of news articles and reports. We offer services such as:

  • Credibility Assessment
  • Media Literacy Workshops
  • Customized Consultation

Working

Our system employs machine learning techniques to classify news articles as either true or fake. Users can input news text, which is then processed and analyzed by our trained model to determine its authenticity.

Key Features

  • News Classification: Classifies news articles as true or fake using a trained machine learning model.
  • Web Interface: User-friendly interface for inputting news text and receiving classification results.
  • Model Evaluation: Performance metrics like accuracy and classification reports.
  • Team and About Pages: Information about the team and project details.

Advantages

  • Quick and accurate assessment of news authenticity.
  • Helps users avoid spreading misinformation.
  • Promotes media literacy and critical thinking.

Limitations

  • Model accuracy depends on the quality and quantity of training data.
  • May not be effective against sophisticated or novel forms of fake news.
  • Requires regular updates and maintenance to remain effective.

Future Scope

  • Enhance model accuracy with advanced machine learning techniques.
  • Expand the web interface with user authentication and news article storage.
  • Incorporate real-time news feeds and automated analysis.

Step-by-Step Guide to Use Our FND System

  1. Clone the repository.
  2. Install required libraries using `pip install -r requirements.txt`.
  3. Run the Flask application using `python app.py`.
  4. Access the application at `http://127.0.0.1:5000`.
